## ✅ بعد التشغيل بنجاح

1. اذهب إلى لوحة التحكم:
   ```
   http://localhost:3000/dashboard/stores/main
   ```

2. ستجد جميع المنتجات والفئات موجودة في:
   - قسم **المنتجات** (Products)
   - قسم **الفئات** (Categories)

3. يمكنك تعديل أو حذف أي منتج من لوحة التحكم

## ⚠️ ملاحظات مهمة

- **لا يتم التحقق من التكرار**: إذا قمت بتشغيل السكريبت مرتين، سيتم إضافة بيانات مكررة
- **الصور**: الصور الحالية هي صور placeholder من Placeholder.com
- **الأسعار**: الأسعار والكميات عشوائية لأغراض الاختبار فقط

## 🧹 لحذف البيانات (إذا لزم الأمر)

إذا أردت حذف جميع المنتجات والفئات للمتجر "main"، يمكنك استخدام:

```javascript
// في MongoDB compass أو mongosh
use lumora-saas

// حذف جميع المنتجات
db.products.deleteMany({ storeId: ObjectId("..") })

// حذف جميع الفئات
db.categories.deleteMany({ storeId: ObjectId("..") })
```
